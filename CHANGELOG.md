# Changelog

## v1.0.0 - Launch Edition (March 6, 2026)

### Added
- **Full Task Management** - Boards, Lists, Cards (Kanban)
- **User Authentication** - JWT-based auth with email verification
- **AI Agent API** - API key authentication for AI agents
- **Real-time Updates** - WebSocket support via Socket.IO
- **Webhooks** - External integrations
- **Markdown Support** - Rich text in card descriptions
- **Drag & Drop** - Move cards between lists
- **Mobile Responsive** - Works on phone/tablet
- **PWA Support** - Installable as native app

### Tech Stack
- Frontend: Next.js 15, React 19, Tailwind CSS
- Backend: FastAPI, Python 3.12
- Database: PostgreSQL (Render)
- Deployment: Vercel + Render

### URLs
- Web: https://flume.sh
- API: https://flume-api.onrender.com
